假设检验与方差分析

参数估计问的是”θ 最可能等于几”。但观测者还有一种截然不同的提问方式：θ 等于某个特定的数吗？两组的均值真的不同吗？看到的效应，是真实的信号还是随机的噪声？本章回答：观测者如何用统计量给信念做”压力测试”。

观测者的困境：效应还是噪声？

回到 Ch06 的观测者。他有一批数据 \(\{X_1, \ldots, X_n\}\)，相信背后有某个参数 \(\theta\) 在驱动一切。上一章，他用似然函数逆向追问：\(\theta\) 最可能是多少？这是估计问题——答案是一个数（或一个区间）。

但观测者还有一种同样根本的提问方式。他不是问”\(\theta\) 有多大”，而是问：“我有理由相信 \(\theta\) 不等于某个特定值吗？”

一颗新药是否比安慰剂有效？两组学生的平均成绩是否有本质差异？一枚骰子是否公平？这些问题共享同一个结构：观测者先有一个默认信念（药无效、两组一样、骰子均匀），然后问——手里的数据是否构成足够强的反证，迫使他放弃这个信念。

这就是假设检验（Hypothesis Testing）要解决的问题。它与估计互补——估计给出了参数的”最佳猜测”，检验则审视这个猜测与某个基准值之间的差距，是否大到不能用随机波动来解释。

检验和估计共享同一套数学对象（统计量、抽样分布、似然函数），但提问方向不同。估计是”量”的问题——参数等于几？检验是”质”的问题——参数是否等于某个特定值？两者合在一起，构成了观测者认知世界的一双眼睛。

但”是否等于某个特定值”这个问题，本身就有两种完全不同的提问方式。这两种方式，分别对应 20 世纪统计学最深刻的范式分裂。

Fisher 的归谬法：显著性检验

R. A. Fisher 在 1925 年出版的《研究工作者的统计方法》中，提出了一个极其朴素但深刻的逻辑。

观测者手里有一个原假设 \(H_0\)——它代表了”一切照常”：药和安慰剂没有区别、骰子是均匀的、施肥对产量没有影响。Fisher 的核心洞见是：

我们只关心 \(H_0\)。我们不需要定义”备择假设”是什么。如果 \(H_0\) 为真时，观测到当前数据（或更极端数据）的概率非常小，那么有两种可能——要么 \(H_0\) 为真而我们运气极差，要么 \(H_0\) 不成立。当这个概率足够小时，我们倾向于后一种解释。

这是一种归谬法（Proof by Contradiction）的概率版本。在经典逻辑中，归谬法断言：如果假设 A 推出矛盾，则 A 不成立。Fisher 的概率归谬法调整了一步：如果假设 A 下，某事件的概率极低——低到你不愿意相信这种”巧合”——则你倾向于否定 A。

这个概率，就是 \(p\) 值（p-value）：

\[p = P(\text{数据} \geq \text{观测到的极端程度} \mid H_0)\]

Fisher 对 \(p\) 值的理解需要特别强调：\(p\) 值衡量的是证据的强度，而不是二元的决策规则。\(p = 0.04\) 意味着”如果 \(H_0\) 为真，只有 4% 的重复实验会产生比当前数据更极端的偏离”。这是一个连续的证据标尺——\(p = 0.001\) 的证据强于 \(p = 0.04\)。Fisher 建议 \(p < 0.05\) 作为”值得报告”的门槛，但他反复强调这只是经验惯例，不是金科玉律。

Fisher 范式的核心特征：只存在 \(H_0\)（没有 \(H_1\)）；\(p\) 值是连续的证据强度度量；不涉及”接受” \(H_0\)（只能”不拒绝”或”拒绝”）；不做长期错误率控制——每次检验是对当前数据的一次独立判断。

从信息论的角度看，Fisher 的 \(p\) 值回答的是：在 \(H_0\) 的假设世界里，我手中的数据有多”令人惊讶”？ 惊讶程度越小，证据越弱；惊讶程度越大，证据越强。这完全是观测者面对单次数据时的认知判断——不涉及长期频率、不涉及决策损失。

Neyman-Pearson 的博弈论：假设检验作为决策规则

Jerzy Neyman 和 Egon Pearson 在 1928-1933 年间提出了一个完全不同的框架。他们不满足于 Fisher 的”证据强度”叙事——在他们看来，统计检验的最终目的是行动：接受还是拒绝？这个决定有后果。

因此他们引入了三个 Fisher 范式中不存在的元素：

第一：备择假设 \(H_1\)。Fisher 只谈 \(H_0\)——你只需要知道原假设是否与数据兼容。NP 指出：如果你要做一个决定，你必须知道替代方案是什么。拒绝 \(H_0\) 意味着接受 \(H_1\)——所以 \(H_1\) 必须被显式定义。

第二：两类错误作为决策风险。 任何决定都可能出错，NP 框架将错误分为两种：

  • 第 I 类错误（\(\alpha\)）：\(H_0\) 为真时拒绝了它——“虚惊一场”。 \[\alpha = P(\text{拒绝 } H_0 \mid H_0 \text{ 为真})\]

  • 第 II 类错误（\(\beta\)）：\(H_1\) 为真时没有拒绝 \(H_0\)——“错失信号”。 \[\beta = P(\text{不拒绝 } H_0 \mid H_1 \text{ 为真})\]

第三：势函数（Power Function）。一个检验在 \(H_1\) 为真时正确拒绝 \(H_0\) 的概率： \[\text{Power}(\theta) = 1 - \beta(\theta) = P(\text{拒绝 } H_0 \mid \theta \in \Theta_1)\]

势是检验的”侦测能力”——效应确实存在时，检验能揪出它的概率。

从这三个元素出发，NP 框架将假设检验重构为一个决策优化问题：

在固定 \(\alpha \leq \alpha_0\)（控制第 I 类错误上限）的约束下，寻找使势 \(1-\beta\) 最大化的拒绝域。即：控制”虚惊”的概率，同时最大化”发现真实效应”的概率。

这与 Fisher 的范式有本质不同。Fisher 问：“数据提供了多强的证据反对 \(H_0\)？”NP 问：“我应该采用什么决策规则，使得在长期重复中，我的错误率被控制在可接受的水平？”

NP 范式的核心特征：显式定义 \(H_1\)；\(\alpha\) 和 \(\beta\) 是长期频率保证——不是单次证据的度量；检验的结果是”接受”或”拒绝”的二元决策；最优检验由 Neyman-Pearson 引理给出（似然比检验是一致最优检验，当它存在时）。

两类错误的认知预算逻辑：固定样本量 \(n\) 下，\(\alpha\) 和 \(\beta\) 不能同时减小——这是 Neyman-Pearson 引理的必然推论，也是 Ch04”信息不是免费的”原则在假设检验中的体现。要同时降低两类错误，唯一的办法是增加样本量——用更多的数据购买更多的确定性。每个检验本质上是一个认知预算分配问题：你更害怕哪种错误？药物审批中，\(\alpha\)（批准无效药）的代价是公共健康风险，\(\beta\)（错过有效药）的代价是患者失去治疗机会——哪个更严重，决定了你如何倾斜 \(\alpha/\beta\) 的天平。这就是为什么显著性水平 0.05 不是宇宙常数——它是特定领域对两类错误相对成本的编码。

专栏：现代假设检验的混合范式

今天翻开任何一本统计教材，你看到的是 Fisher 和 NP 框架的杂糅体：

  1. 设定 \(H_0\) 和 \(H_1\)（NP 框架）
  2. 选择显著性水平 \(\alpha\)（NP 框架——控制第 I 类错误）
  3. 计算 \(p\) 值（Fisher 框架——衡量证据强度）
  4. 如果 \(p < \alpha\)，拒绝 \(H_0\)（用 NP 的决策规则解释 Fisher 的连续证据度量）

这是一种理论上的不协调。Fisher 终其一生反对 NP 框架——他认为”备择假设”“两类错误”“势函数”把统计推断变成了机械的工业决策规则，丧失了科学推断的灵活性和判断力。NP 则批评 Fisher 的 \(p\) 值缺乏决策理论基础——你拿到 \(p = 0.04\)，然后呢？行动还是不行动？标准在哪？

但为什么现代实践变成了两者的融合？

因为两者各自解决了对方没有解决的核心问题。 Fisher 提供了直觉上极其自然的证据度量（\(p\) 值）——观测者天生想知道”我的数据有多极端”。NP 提供了决策必需的数学框架（\(\alpha\) 控制、势分析）——当观测者必须基于数据做决定时，他需要知道决定的长期错误率。两者的结合虽然在哲学上不纯粹，但在实践上是功能完整的：用 NP 的框架规划实验（确定样本量、控制 \(\alpha\) 和 \(\beta\)），用 Fisher 的 \(p\) 值来报告证据强度（而不是简单地报告”拒绝/不拒绝”）。

对观测者而言，这场融合的实用教训是：报告 \(p\) 值，但不要只报告 \(p\) 值。报告效应量（Ch07 §7.7），报告置信区间（Ch06），报告势分析。统计推断不是一台产出”显著/不显著”二值标签的机器——它是观测者在不确定下做出判断的完整认知过程。

从 Ch08 的贝叶斯视角看，Fisher 和 NP 的争论还有一个更深的维度：两者都基于 \(P(\text{数据} \mid H_0)\)——“如果原假设为真，这些数据的概率是多少？”贝叶斯学派直接计算 \(P(H_0 \mid \text{数据})\)——“给定这些数据，原假设为真的概率是多少？”前者回答关于数据的频率，后者回答关于假设本身的信念。两种回答各有其适用场景，但问的是不同的问题。

检验统计量：从哪里来，到哪里去

铺垫了两种检验哲学后，回到数学本身。无论 Fisher 还是 NP 范式，都需要一个核心工具：检验统计量。

直观起源

Ch05 定义了统计量为”不含未知参数的样本函数”——它将 \(n\) 维的原始数据压缩为低维摘要。检验统计量是统计量的一个子类——但它的诞生有一种额外的动机。

考虑最简单的场景：观测者想知道总体均值 \(\mu\) 是否等于 \(\mu_0\)。他的直觉是：如果样本均值 \(\bar{x}\) 离 \(\mu_0\) “很远”，\(H_0\) 就值得怀疑。但”远”本身不构成判断——\(\bar{x}\) 的分布在不同的 \(\sigma\) 下有不同的尺度。所以观测者必须把距离标准化：

\[T = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}\]

这个 \(T\) 就是检验统计量。它的构建逻辑是：

  1. 选定一个能够区分 \(H_0\) 和 \(H_1\) 的量：\(\bar{x} - \mu_0\)——在 \(H_0\) 下它应该在 0 附近，在 \(H_1\) 下它倾向于偏离 0。
  2. 用它的标准误做分母，消除尺度的不确定性：除以 \(\sigma/\sqrt{n}\)——这是 \(\bar{x}\) 在 \(H_0\) 下的标准差。
  3. 结果是一个”在 \(H_0\) 下分布完全已知”的量：\(T \sim N(0,1)\)。这使观测者可以精确地计算 \(p\) 值或确定拒绝域。

检验统计量的本质，是一个”信噪比”的雏形。 分子是观测到的效应（偏离 \(H_0\) 的程度），分母是该效应在随机波动下的标准尺度。当信噪比足够大时，观测者判断：效应不太可能纯由噪声产生。方差分析的 \(F\) 统计量将把这个逻辑系统化到极致。

从充分统计量到检验统计量

Ch05 的因子分解定理告诉我们：充分统计量 \(T\) 包含了样本中关于 \(\theta\) 的全部信息。这个事实对检验统计量的构造有直接影响：

  • 如果充分统计量存在，检验统计量应该——而且通常就是——充分统计量的函数。 这保证了检验不会丢弃数据中与 \(\theta\) 相关的信息。例如，正态总体的 \(\bar{x}\) 是 \(\mu\) 的充分统计量——所有关于 \(\mu\) 的检验都以 \(\bar{x}\) 为出发点。
  • 似然比检验（Likelihood Ratio Test）是这一原则的终极表达：检验统计量 \(\Lambda = \frac{\sup_{\Theta_0} L(\theta)}{\sup_{\Theta} L(\theta)}\) 自动地基于充分统计量构造——因为似然函数本身经由充分统计量流通（因子分解定理）。

从信息论的角度看，检验统计量是一个信息通道的瓶颈：样本中的所有信息通过它流向最终的判断。一个好的检验统计量，必须在 \(H_0\) 和 \(H_1\) 下表现出系统性的分布差异——否则这个通道无法区分两个假设。

正态总体下的检验工具箱

前一节讲的是检验统计量的”为什么”。这一节给出具体场景下的”怎么做”——观测者在正态假设下的标准检验工具。

关于均值的检验

单个总体，\(\sigma\) 已知——\(u\) 检验。 这是最简单的情况：观测者知道数据的方差（稀有，但在工业质量控制中偶尔出现）。

\[u = \frac{\bar{x} - \mu_0}{\sigma_0 / \sqrt{n}} \sim N(0,1)\]

双侧检验——观测者没有先验方向预期——的拒绝域为 \(W = \{|u| \geq u_{1-\alpha/2}\}\)。

单个总体，\(\sigma\) 未知——\(t\) 检验。 这才是现实中最常见的情况。用样本标准差 \(s\) 替代 \(\sigma\)：

\[t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}} \sim t(n-1)\]

拒绝域：\(W = \{|t| \geq t_{1-\alpha/2}(n-1)\}\)。

\(t\) 分布在 Ch05 已经登场——它的厚尾是对”用 \(s\) 估计 \(\sigma\) 的额外不确定性”的诚实补偿。样本量越小，这种补偿越显著；\(n \to \infty\) 时 \(t \to N(0,1)\)——足够多的数据最终消解了不确定性。

两个总体，\(\sigma_1, \sigma_2\) 已知——两样本 \(u\) 检验。 观测者比较两组的均值：

\[u = \frac{\bar{x} - \bar{y}}{\sqrt{\frac{\sigma_1^2}{m} + \frac{\sigma_2^2}{n}}} \sim N(0,1)\]

两个总体，方差相等但未知——两样本 \(t\) 检验。 这是”两组对比”的标准工具。合并方差：

\[s_w^2 = \frac{1}{m+n-2}\left[\sum_{i=1}^m (x_i - \bar{x})^2 + \sum_{i=1}^n (y_i - \bar{y})^2\right]\]

\[t = \frac{(\bar{x} - \bar{y}) - (\mu_1 - \mu_2)}{s_w \sqrt{\frac{1}{m} + \frac{1}{n}}} \sim t(m+n-2)\]

非正态总体，大样本——渐近 \(u\) 检验。 当总体不是正态分布但样本足够大时，CLT（Ch03）接管：

\[u = \frac{\bar{x} - \bar{y}}{\sqrt{\frac{s_x^2}{m} + \frac{s_y^2}{n}}} \overset{\text{approx}}{\sim} N(0,1)\]

小样本下的近似 \(t\) 检验使用 Welch-Satterthwaite 自由度：

\[l = \frac{\left(\frac{s_x^2}{m} + \frac{s_y^2}{n}\right)^2}{\frac{s_x^4}{m^2(m-1)} + \frac{s_y^4}{n^2(n-1)}}\]

成对数据检验。 如果数据是成对出现的（同一个体处理前后、同一块田分成两半），两样本 \(t\) 检验不适用——因为它忽略了配对结构。正确的做法是对差值做单样本 \(t\) 检验：\(d_i = x_i - y_i\)，然后检验 \(H_0: \mu_d = 0\)。

关于方差的检验

单个正态总体方差——\(\chi^2\) 检验。 观测者想知道总体的波动程度是否等于某个标称值：

\[\chi^2 = \frac{(n-1)s^2}{\sigma_0^2} \sim \chi^2(n-1)\]

双侧拒绝域：\(W = \{\chi^2 \leq \chi^2_{\alpha/2}(n-1)\} \cup \{\chi^2 \geq \chi^2_{1-\alpha/2}(n-1)\}\)。

两个正态总体方差比——\(F\) 检验。 比较两组数据的波动程度：

\[F = \frac{s_x^2}{s_y^2} \sim F(m-1, n-1)\]

这个 \(F\) 统计量将在 ANOVA 中扮演更宏大的角色——不只是比较两个方差，而是将总变异拆解为”信号”和”噪声”两部分。

指数分布参数的检验

对于指数分布 \(X \sim \text{Exp}(1/\theta)\)，检验 \(H_0: \theta \leq \theta_0\) 可利用指数分布与 \(\chi^2\) 分布的精确关系：

\[\chi^2 = \frac{2n\bar{x}}{\theta_0} \sim \chi^2(2n)\]

大样本检验的一般框架

设总体均值为 \(\theta\)，方差为 \(\sigma^2(\theta)\)（方差不独立，而是 \(\theta\) 的函数——这在非正态总体中很常见）。由 CLT：

\[u = \frac{\sqrt{n}(\bar{x} - \theta_0)}{\sqrt{\sigma^2(\hat{\theta})}} \overset{\text{approx}}{\sim} N(0,1)\]

其中 \(\hat{\theta}\) 是 \(\theta\) 的 MLE（Ch06）。这个框架极其通用——它使观测者在”不知道精确分布但样本量大”的情况下依然能做检验。但大样本检验有一个隐蔽的陷阱——见 §7.7。

方差分析（ANOVA）

前面所有的检验都是”两组比较”——观测者问两个均值是否相等。但现实中，观测者常常面对多个组：四种肥料哪种最好？五种教学方法是否有差异？三个地区的收入水平是否相同？

最朴素的做法是两两做 \(t\) 检验。但如果有三组，就需要做 \(\binom{3}{2} = 3\) 次检验；四组需要 6 次；五组需要 10 次。每次检验都有自己的第 I 类错误率 \(\alpha\)——多次检验的累积错误率会远超 \(\alpha\)。而且两两比较无法回答一个更根本的问题：整体上，这些组间是否有差异？

Fisher 在 1920 年代的农业试验站工作中解决了这个问题。他的解法是方差分析（ANOVA）——一个看似在做”方差比较”、实则在检验”均值差异”的巧妙构造。

核心直觉：信号 + 噪声 = 总量

观测者面前摆着 \(k\) 个组的观测值。任何单个观测值 \(x_{ij}\)（第 \(i\) 组的第 \(j\) 个观测）可以分解为三个部分：

\[x_{ij} = \mu + \alpha_i + \epsilon_{ij}\]

  • \(\mu\)：总均值——所有组共享的基线。
  • \(\alpha_i\)：第 \(i\) 组的处理效应——组均值偏离总均值的程度。如果各组真的没有差异，所有 \(\alpha_i = 0\)。
  • \(\epsilon_{ij}\)：随机误差——无法解释的噪声。

ANOVA 的逻辑由此展开。总变异被拆成两个来源：

\[\underbrace{\sum_{i=1}^k \sum_{j=1}^{n_i} (x_{ij} - \bar{x})^2}_{\text{SST（总平方和）}} = \underbrace{\sum_{i=1}^k n_i (\bar{x}_i - \bar{x})^2}_{\text{SSA（组间平方和 = 信号）}} + \underbrace{\sum_{i=1}^k \sum_{j=1}^{n_i} (x_{ij} - \bar{x}_i)^2}_{\text{SSE（组内平方和 = 噪声）}}\]

SST = SSA + SSE——这不是一个近似，而是一个精确的代数恒等式。如果 \(H_0\) 为真（各组均值相等），SSA 和 SSE 都是”噪声”——两者的期望都是方差的无偏估计。SSA 只是在 \(H_0\) 为真时碰巧膨胀——但它的膨胀程度被限制在纯随机波动的范围内。

但如果各组均值真的有差异，SSA 会被”信号”膨胀——它不再只是噪声。ANOVA 检验统计量 \(F\) 的构造直指这个直觉：

\[F = \frac{\text{SSA} / (k-1)}{\text{SSE} / (n-k)} = \frac{\text{MSA}}{\text{MSE}} = \frac{\text{信号}}{\text{噪声}}\]

\(F\) 统计量是信噪比的精确数学表达。 分子 MSA 包含”信号 + 噪声”，分母 MSE 只包含”噪声”。如果 \(H_0\) 为真（无信号），\(F\) 在 1 附近波动；如果 \(H_1\) 为真（有信号），分子膨胀，\(F\) 远大于 1。ANOVA 没有检验”均值是否相等”——它分离了信号的贡献，再与噪声的基准做对比。这就是为什么它被称为信噪比分离器。

在 \(H_0\) 为真时，\(F \sim F(k-1, n-k)\)。拒绝域是右侧单边的：\(W = \{F > F_{1-\alpha}(k-1, n-k)\}\)——只有信号显著强于噪声时，观测者才宣告效应存在。

单因素方差分析表——信噪比的会计账本
误差来源 平方和 自由度 均方（MS） \(F\) 值
处理效应（信号） SSA \(k-1\) \(\text{MSA} = \text{SSA}/(k-1)\) \(F = \text{MSA}/\text{MSE}\)
误差（噪声） SSE \(n-k\) \(\text{MSE} = \text{SSE}/(n-k)\)
总变异 \(\text{SST} = \text{SSA}+\text{SSE}\) \(n-1\)

ANOVA 的假设（正态性、方差齐性、独立性）不是装饰。如果方差不齐——各组内部的噪声水平不同——“信噪比”的分母就不再是一个干净的噪声基线，\(F\) 检验的可靠性随之瓦解。独立性被破坏（如重复测量）时，数据的冗余会人为压低噪声估计，导致虚假的高 \(F\) 值。观测者在使用 ANOVA 前，必须审视这些假设——正如在 Ch10 的回归分析中审视残差。

警示：当 \(\alpha\) 失效——大数据时代的显著性危机

前面六节建立了一个优雅的假设检验体系。但观测者必须面对一个不容回避的现代困境：这个体系在大数据时代面临根本性的张力。

样本量无限膨胀时，一切差异都”显著”

\(p\) 值的定义包含了样本量——它不直接出现在 \(p\) 值的公式中，但所有的检验统计量都含有 \(\sqrt{n}\) 因子。考虑最简单的 \(u\) 检验：

\[u = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}} = \sqrt{n} \cdot \frac{\bar{x} - \mu_0}{\sigma}\]

固定效应量 \(\frac{\bar{x} - \mu_0}{\sigma}\) 不变。当 \(n \to \infty\) 时，\(|u| \to \infty\)，\(p \to 0\)。

这意味着：只要真实效应不完全为零——哪怕它微小到在实践上毫无意义——足够大的样本量都能让 \(p\) 值降到任意低的水平。 如果 \(\mu\) 的真实值是 100.0001 而 \(H_0\) 说 \(\mu = 100\)，当 \(n = 10^8\) 时，\(p\) 值会远低于 0.05。统计上显著，但偏离微乎其微——0.0001 的差距在大多数应用场景中没有任何实际意义。

这不是假设检验的 bug——这是它作为数学工具的内禀特征。\(p\) 值的设计目标从来不是衡量”效应有多大”，而是衡量”数据中有多强的证据反对 \(H_0\)“。它是证据强度的量表，不是效应大小的量表。但在大数据环境下，这两个概念的脱钩变得极其危险。

效应量：观测者真正该关心的数字

如果 \(p\) 值不能告诉你效应有多大——什么能？

效应量（Effect Size）是对”效应有多大”的标准化度量。它不依赖于样本量，只描述现象本身的规模：

  • Cohen’s \(d\)（两组均值差除以合并标准差）：\(d = \frac{\bar{x}_1 - \bar{x}_2}{s_{\text{pooled}}}\)。\(d = 0.2\) 是”小”效应，\(d = 0.5\) 是”中”效应，\(d = 0.8\) 是”大”效应。
  • \(\eta^2\)（ANOVA 中的效应量）：\(\eta^2 = \frac{\text{SSA}}{\text{SST}}\)——信号在总变异中的占比。\(\eta^2 = 0.01\) 意味着信号只解释了 1% 的变化，即使 \(p < 0.001\) 也不能说明效应”重要”。
  • 相关比 \(\omega^2\)、Cramér’s \(V\)、优势比（Odds Ratio）——每种检验都有对应的效应量度量。

统计显著性 \(\neq\) 实际显著性。前者回答：“观测到的差异能否被纯随机波动解释？”（\(p\) 值）。后者回答：“观测到的差异在现实中足够大，以至于值得关注吗？”（效应量）。\(p\) 值是对信噪比的检验——样本量越大，你能区分越微弱的信号。效应量是对信号本身大小的度量——它与样本量无关。

观测者的完整检验清单

面对现代数据环境，观测者应该在每次检验后问自己三个问题，而非一个：

  1. \(p\) 值告诉我有证据反对 \(H_0\)（证据有多强？）
  2. 效应量告诉我效应有多大（大到值得关心吗？）
  3. 置信区间告诉我效应的合理范围（我们对效应大小的认知有多精确？）

三者缺一不可。\(p\) 值给出证据的存在性，效应量给出效应的量级，置信区间给出估计的精度——合在一起，才是观测者对效应现象的完整认知。

这一警示不是要抛弃假设检验——它是统计推断不可缺少的工具。但观测者不能把它当作自动化的”显著/不显著”标签机。统计推断的终点不是 \(p < 0.05\)——是理解数据背后的世界。

速查卡片：

  • 核心问题：观测到的效应是真实信号还是随机噪声？多组均值是否有显著差异？
  • 两种范式：Fisher（\(p\) 值 = 连续证据强度，归谬法）vs. Neyman-Pearson（\(\alpha\)/\(\beta\) 控制，决策规则，势函数）→ 现代实践 = 两者的”包办婚姻”
  • 检验统计量：构建逻辑 = (效应估计 - 假设值) / (效应估计的标准误)——在 \(H_0\) 下分布完全已知的信噪比雏形
  • 关键概念：\(p\) 值 → \(\alpha\)（显著性水平）→ 两类错误 → 势函数 → \(u\) 检验 / \(t\) 检验 / \(\chi^2\) 检验 / \(F\) 检验 → 方差分析（ANOVA）：\(\text{SST} = \text{SSA} + \text{SSE}\)，\(F = \text{MSA}/\text{MSE}\) = 信号/噪声
  • 现代警示：大样本下一切差异都”显著” → \(p\) 值 ≠ 效应大小 → 始终报告效应量 + 置信区间
  • R 命令：t.test(), var.test(), aov(), anova(), oneway.test(), pairwise.t.test(), effectsize::cohens_d(), effectsize::eta_squared()

深度思考：

  1. Fisher 说”我们只关心 \(H_0\)“，NP 说”必须有 \(H_1\)“。从信息论的角度看，两者的分歧在于：检验的”信息输出”应该是什么？Fisher 输出的是一个连续的证据度量（\(p\) 值——“有多少信息反对 \(H_0\)”），NP 输出的是一个二元决策（“接受/拒绝”——“是否行动”）。在什么场景下，连续证据优于二元决策？在什么场景下，二元决策是必需的？

  2. \(t\) 检验的分子是 \(\bar{x} - \mu_0\)，分母是 \(s/\sqrt{n}\)。为什么分母是 \(s/\sqrt{n}\) 而不是 \(s\)？——提示：回忆 Ch03 中样本均值的标准差。如果 \(H_0\) 为真，但观测者误用了错误的 \(\sigma\)（比如以为是 \(\sigma_0\) 而实际上是 \(2\sigma_0\)），\(p\) 值会受到什么影响？这对应着第 I 类错误率的膨胀还是检验势的降低？

  3. ANOVA 的 \(F\) 检验是右侧单边的——拒绝域只在 \(F\) 很大的情况下触发。为什么 \(F\) 值很小（远小于 1）时不构成反对 \(H_0\) 的证据？如果 \(F\) 异常地小（比如 \(F \approx 0.01\)），这可能暗示着数据的什么问题？——提示：思考 MSE 和 MSA 在什么情况下会异常地接近零，以及这与 ANOVA 的独立性假设有何关系。

  4. 在大数据场景下，研究者可能同时检验数百个假设（基因组学中的全基因组关联分析、电商平台上的 A/B 测试矩阵）。单次检验的 \(\alpha = 0.05\) 意味着每 20 次检验平均有 1 次虚惊。如果同时做 1000 次独立检验，至少出现一次虚惊的概率是多少？这引出了多重检验校正（如 Bonferroni、FDR）的必要性——它与 Ch04 的联合界不等式（Union Bound）有何联系？在什么意义上，多重检验校正是”信息不是免费的”的下一个层次——不仅单个检验需要信息，整个检验家族的可靠性也需要信息（样本量或更严格的阈值）来维护？

  5. 效应量（Cohen’s \(d\) 等）不依赖于样本量——它衡量的是总体现象本身的大小。但如果总体效应确实很小（比如 \(d = 0.05\)），需要多大的样本量才能以 80% 的势检测到它？这个计算揭示了”检测小效应”与”研究经费”之间的现实权衡——在什么意义上，势分析本身也是一种认知预算规划？

参数估计

概率论从参数走向数据（演绎）。统计推断从数据走回参数（归纳）。本章回答：这条回头路，怎么走才最少损失信息？又能走到多准？

统计推断本质是逆向工程

回忆 Ch05 的开篇：

\[\text{概率论（演绎）}: \theta \longrightarrow X \qquad \text{数理统计（归纳）}: X \longrightarrow \theta\]

现在把这两行放在同一个数学对象——似然函数——下观察。当 \(\theta\) 已知时，\(p(x;\theta)\) 描述数据 \(x\) 如何从参数 \(\theta\) “演化”出来。这个演化过程是概率论的正向计算。而统计推断的任务是逆转这个箭头：观测到了 \(x\)，固定 \(x\)，把 \(\theta\) 当作变量——让密度函数 \(p(x;\theta)\) 反过来”审问”每一个可能的 \(\theta\)：在这个 \(\theta\) 下，出现我手中这批数据的概率有多大？

这就是似然（Likelihood）的逻辑。它与概率共享同一个函数形式 \(p(x;\theta)\)，但箭头方向完全相反。样本是 \(n\) 维空间中的一个点——一个高度冗余的高维信号。参数 \(\theta\) 只有区区 \(k\) 维（\(k \ll n\)）。参数估计的任务，本质上是从 \(n\) 维投影逆向还原 \(k\) 维投影源——一个典型的逆问题（Inverse Problem）。

本章用五步走完这条回头路。第一步：用什么标准评判估计的好坏？第二步：如何对估计做”无损提纯”？第三步：估计精度的物理极限在哪里？第四步：谁最接近这个极限？第五步：如果放弃模型假设，还有什么备选方案？

估计的优良性

给定样本 \(X_1, \ldots, X_n\)，一个估计量 \(\hat{\theta} = \hat{\theta}(X_1, \ldots, X_n)\) 是对未知参数 \(\theta\) 的”猜测”。但猜测有好有坏——需要一个评判尺度。

无偏性

无偏性（Unbiasedness）是最朴素的要求：在重复抽样的意义上，估计量不应该系统性地偏向任何一方。

\[E_\theta(\hat{\theta}) = \theta, \quad \forall \theta \in \Theta\]

样本均值 \(\bar{X}\) 是 \(\mu\) 的无偏估计——无论抽样多少次，\(\bar{X}\) 的平均命中靶心。但样本方差 \(s_n^2 = \frac{1}{n}\sum (X_i - \bar{X})^2\) 不是 \(\sigma^2\) 的无偏估计：\(E(s_n^2) = \frac{n-1}{n}\sigma^2\)。偏差来源于 \(\bar{X}\) 消耗了一个自由度去估计 \(\mu\)——\(n\) 个偏差 \((X_i - \bar{X})\) 被一个线性约束 \(\sum (X_i - \bar{X}) = 0\) 捆绑，真正独立的只有 \(n-1\) 个。除以 \(n-1\) 的无偏版本 \(s^2\) 是对”用同一批数据先估 \(\mu\) 再估 \(\sigma^2\)“这一双重使用的诚实补偿。

无偏性保障了”群体公平”——如果你把全世界的统计学家各自独立做的估计收集起来，它们的平均是正确的。但它不回答：你手里的这一次估计，离真相有多远？

均方误差：偏差-方差分解

一个估计量即使无偏，波动可能极大——每次都”平均正确”，但单次可以偏得离谱。均方误差（Mean Squared Error, MSE）同时惩罚偏差和波动：

\[\text{MSE}(\hat{\theta}) = E[(\hat{\theta} - \theta)^2] = \text{Var}(\hat{\theta}) + \underbrace{(E[\hat{\theta}] - \theta)^2}_{\text{Bias}^2}\]

这是一个会计恒等式——总误差 = 方差 + 偏差²。两个分量之间存在此消彼长的权衡（Bias-Variance Trade-off）：

  • 无偏估计：偏差为零，但方差可能很大。模型对数据的每一次随机波动都忠实响应——包括那些纯噪声的波动。
  • 有偏估计：主动引入一点系统性偏差，换取方差的大幅降低。你不再对数据的每一次微小变化一惊一乍——估计变得”稳重”了，尽管它不再保证平均正确。

为什么有时愿意忍受偏差？ 因为 MSE 是平方损失——方差直接进入总误差，而偏差以平方进入。如果你可以牺牲 \(0.1\) 的偏差来换取方差从 \(1.0\) 降到 \(0.01\)，MSE 从 \(1.0\) 跌至 \(0.02\)。这是 Ch10（岭回归、Lasso）中正则化技术的全部逻辑地基：用一点点偏差购买大量的稳定性。

充分性原则

Ch05 建立了充分统计量的概念——\(T(X)\) 包含了样本中关于 \(\theta\) 的全部信息。现在这个概念的统计推断含义浮现出来。

充分性原则：如果 \(T\) 是 \(\theta\) 的充分统计量，那么任何统计推断都可以——而且应该——只基于 \(T\) 进行。在参数估计的语境下，这一原则催生了 Rao-Blackwell 定理：

设 \(\hat{\theta}\) 是 \(\theta\) 的任一无偏估计，\(T\) 是充分统计量。定义 \(\tilde{\theta} = E[\hat{\theta} \mid T]\)。则 \(\tilde{\theta}\) 也是无偏估计，且 \(\text{Var}(\tilde{\theta}) \leq \text{Var}(\hat{\theta})\)。

这个定理的操作含义可以用一句话概括：如果你的估计量不是充分统计量的函数，你对充分统计量求一次条件期望，就能得到一个方差更小的估计——且不引入任何偏差。

为什么这一定理成立？直觉如下。\(\hat{\theta}\) 可以分解为两部分：

\[\hat{\theta} = E[\hat{\theta} \mid T] + (\hat{\theta} - E[\hat{\theta} \mid T])\]

第一项是 \(T\) 的函数——它只包含样本中与 \(\theta\) 相关的”信号”。第二项是残差——它的条件期望为 0，且与第一项不相关。这个残差就是 \(\hat{\theta}\) 中混入的纯噪声——它的存在只增加方差，不贡献任何关于 \(\theta\) 的信息。求条件期望 \(E[\cdot \mid T]\) 的操作，本质上是一次信号提纯：它将 \(\hat{\theta}\) 投影到充分统计量 \(T\) 生成的 \(\sigma\)-代数上，滤掉了所有与 \(\theta\) 正交的噪声成分。

充分性 + Rao-Blackwell = 统计学中的”炼金术”：不需要额外的数据，不需要更强的假设——仅通过对充分统计量做一次条件期望，就能从含杂质的原始估计中提纯出方差更小的版本。这也是为什么一致最小方差无偏估计（UMVUE）——当它存在时——必然可以表示为充分统计量的函数。

Fisher 信息量与 C-R 下界

Rao-Blackwell 告诉你可以不断降低方差。但方差能降到多低？是否存在一个任何无偏估计都无法突破的物理极限？

Fisher 信息量：似然函数的曲率

设对数似然函数为 \(\ell(\theta) = \log p(x;\theta)\)。它的一阶导数 \(\ell'(\theta)\) 称为得分函数（Score Function）——衡量对数似然在 \(\theta\) 处的瞬时变化率。Fisher 信息量定义为得分函数的方差：

\[I(\theta) = \text{Var}_\theta\left(\frac{\partial}{\partial\theta} \log p(X;\theta)\right) = -E_\theta\left[\frac{\partial^2}{\partial\theta^2} \log p(X;\theta)\right]\]

第二个等号（期望的二阶导的负数）赋予了 Fisher 信息一个关键的几何直觉。\(\ell(\theta)\) 在最大值处是上凸的——二阶导为负。Fisher 信息量就是对数似然函数在其峰值附近的平均曲率。曲率越大，似然峰越尖锐，“最可能的 \(\theta\)”和”次可能的 \(\theta\)“的区分越清晰——数据中关于 \(\theta\) 的信息越集中。

Fisher 信息不是”数据包含多少信息”，而是”数据中关于 \(\theta\) 的信息有多尖锐地集中在一个峰周围”。一个扁平的似然函数意味着多种 \(\theta\) 值对数据的解释能力几乎相同——你对 \(\theta\) 的认知是模糊的。

C-R 不等式：精度的物理下界

设 \(T\) 是 \(g(\theta)\) 的任一无偏估计。在正则条件下：

\[\text{Var}(T) \geq \frac{[g'(\theta)]^2}{n I(\theta)}\]

特别地，对 \(\theta\) 本身的无偏估计（\(g(\theta) = \theta\)，\(g'(\theta) = 1\)）：

\[\text{Var}(\hat{\theta}) \geq \frac{1}{n I(\theta)}\]

这就是 Cramér-Rao 下界——统计学中的”测不准原理”。它声明：无论你多么聪明地构造估计量，其方差不可能低于 \(1/(nI(\theta))\)。 信息是有限的，精度是有天花板的。

达到 C-R 下界的估计称为有效估计（Efficient Estimator）。它不多见——只在指数型分布族中才可能存在。但 C-R 下界的价值不在于”能否达到”，而在于它为所有估计量提供了一个绝对的比较基准：你的估计量的方差离下界还有多远？——这个比值（效率，Efficiency）定义了你的估计方法还有多少改进空间。

最大似然估计

现在来到参数估计的最高点。给定数据，哪个 \(\theta\) 是”最好的”？最大似然估计（MLE）给出的答案极其朴素：选那个最可能产生这批数据的参数。

\[\hat{\theta}_{\text{MLE}} = \arg\max_\theta \prod_{i=1}^n p(x_i; \theta) = \arg\max_\theta \sum_{i=1}^n \log p(x_i; \theta)\]

为什么是对数？

乘积换成求和不仅是计算便利。\(\log p(x_i; \theta)\) 是单个观测 \(x_i\) 在参数 \(\theta\) 下的对数似然——它可以被理解为”\(\theta\) 对 \(x_i\) 的解释质量”。求和意味着：MLE 在选择那个对全体数据解释质量之和最高的参数。这是民主的——每个数据点有一票。

MLE = 最小化 KL 散度

Ch01 定义了 KL 散度 \(D_{KL}(P \parallel Q)\)：用分布 \(Q\) 近似真实分布 \(P\) 时，额外浪费的信息量。现在把这个视角套到估计问题上。设 \(\hat{P}_n\) 为经验分布——在每个观测值 \(x_i\) 处放置 \(1/n\) 的概率质量。\(P_\theta\) 为参数模型。则：

\[ \arg\min_\theta D_{KL}(\hat{P}_n \parallel P_\theta) = \arg\min_\theta \left[-\frac{1}{n}\sum_i \log p_\theta(x_i) - H(\hat{P}_n)\right] = \arg\max_\theta \sum_i \log p_\theta(x_i) = \hat{\theta}_{\text{MLE}} \]

经验分布的熵 \(H(\hat{P}_n)\) 与 \(\theta\) 无关——它消掉了。剩下的恰好是对数似然。

MLE 不是在”猜”参数。它是在对经验分布做有损压缩——用光滑的参数模型 \(P_\theta\) 替代原始的频率阶梯 \(\hat{P}_n\)，并最小化压缩过程中的信息损失。

这个等价性是全书最重要的数学事实之一：MLE = 最小化 KL 散度。它的哲学回响是深远的：

  • Ch08 的贝叶斯 MAP 估计 = MLE + 先验惩罚项——同样是信息损失最小化，只是多了一个”初始信念”的正则化约束。
  • Ch18 决策树在每一步用信息增益选择分裂变量——信息增益就是 KL 散度在分类问题中的化身。
  • Ch20 的交叉熵损失函数 = 经验分布与模型分布之间的交叉熵——它和 KL 散度只差一个常数 \(H(\hat{P}_n)\)。

MLE、决策树、深度学习——三层不同的方法论，共享同一套信息论逻辑。300 年的统计学发展，汇入一条河。

大样本下的完美表现

MLE 不仅是”信息损失最小”的哲学最优解，它在大样本下逼近工程最优：

  1. 相合性：\(\hat{\theta}_n \xrightarrow{P} \theta_0\)（当 \(n \to \infty\)）。数据足够多时，MLE 会无限接近真相——这是 LLN（Ch03）的直接推论。
  2. 渐近正态性：\(\sqrt{n}(\hat{\theta}_n - \theta_0) \xrightarrow{d} N(0, I^{-1}(\theta_0))\)。MLE 的渐近分布是正态——CLT（Ch03）的又一个胜利——且其渐近方差恰好等于 C-R 下界的倒数。在大样本下，MLE 渐近地达到了估计精度的物理极限。
  3. 充分性兼容：MLE 总是充分统计量的函数——它自动继承了充分统计量的全部信息保持特性。

矩估计（下一节）是经验的——不假设分布形式，用样本矩匹配总体矩，简单直接。MLE 是优美的——完全依赖模型假设，在假设成立时达到最优。两者的差异，是”稳健的工程直觉”与”精确的信息论最优”之间的永恒张力。

矩估计的补充

如果观测者不愿意——或没有足够的理由——假设数据来自某个特定的参数族，他还能估计参数吗？

矩估计（Method of Moments）给出了一个不依赖分布假设的备选方案。它的逻辑直截了当：

  1. 总体 \(k\) 阶矩 \(\mu_k = E(X^k)\) 是参数 \(\theta\) 的函数：\(\mu_k = g_k(\theta)\)。
  2. 样本 \(k\) 阶矩 \(m_k = \frac{1}{n}\sum X_i^k\) 是 \(\mu_k\) 的相合估计（由 LLN 保证）。
  3. 因此，用样本矩替换总体矩，反解出 \(\theta\)：\(\hat{\theta} = g^{-1}(m_1, \ldots, m_k)\)。

矩估计本质上是在用经验分布函数 \(\hat{F}_n\) 的前 \(k\) 个矩特征去匹配参数模型的前 \(k\) 个矩特征——它不关心整个分布的形状，只要求两个分布的”粗粒度轮廓”对齐。

矩估计 vs. MLE：

矩估计 MLE
依赖假设 仅需矩存在 需完整的分布形式
最优性 一般达不到 C-R 下界 渐近达到 C-R 下界
稳健性 对模型误设更宽容 对模型误设敏感
计算 解方程组即可 常需数值优化

矩估计是经验主义的：不相信任何分布假设，只用样本的粗粒度特征去反推参数。它的朴实使它成为 MLE 不可或缺的对比基准——当你不知道真实分布时，矩估计至少是相合的；当你知道真实分布时，MLE 是渐近最优的。两者之间的取舍，取决于你对模型假设的信心。

速查卡片：

  • 核心问题：观测到数据 \(\{X_i\}\) 后，如何逆向还原参数 \(\theta\)？能还原到多准？

  • R 命令：MASS::fitdistr(), bbmle::mle2(), fitdistrplus::fitdist(), optim() 自定义似然

深度思考：

  1. Rao-Blackwell 定理说 \(E[\hat{\theta} \mid T]\) 的方差不大于 \(\hat{\theta}\)。这个”不大于”是严格的吗？在什么条件下两者方差相等？——提示：当 \(\hat{\theta}\) 已经是 \(T\) 的函数时，条件期望不改变它。Rao-Blackwell 只有在原始估计量”不充分”时才起作用——这本身就是一个诊断工具：你的估计量离充分性有多远？

  2. Fisher 信息量 \(I(\theta) = -E[\ell''(\theta)]\) 的几何含义是似然函数的平均曲率。这里有一个容易被忽略的前提：期望是对 \(p(x;\theta)\) 取的——即”用真实的 \(\theta\) 生成数据，再计算平均曲率”。如果你在错误的 \(\theta\) 处取期望，得到的曲率是失真的。这对 MLE 的渐近方差估计有何影响？

  3. MLE = 最小化 \(D_{KL}(\hat{P}_n \parallel P_\theta)\)——这是正向 KL。Ch09 的变分推断使用反向 KL \(D_{KL}(Q_\theta \parallel P)\)。两者的不对称性在什么场景下导致截然不同的估计行为？提示：正向 KL 是”均值追踪”（覆盖所有模式），反向 KL 是”模式追踪”（锁定最可能的一个模式）。这一差异在 Ch20 的生成模型中有戏剧性的表现。

  4. C-R 下界是 \(1/(nI(\theta))\)。如果 \(I(\theta) = 0\)（似然函数在 \(\theta\) 处是平坦的），下界发散——参数在信息论意义上不可估。什么情况下 \(I(\theta) = 0\)？这意味着数据中真的没有关于 \(\theta\) 的信息，还是参数化本身出了问题——两个不同的 \(\theta\) 产生了完全相同的分布？

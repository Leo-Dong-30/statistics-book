多维随机变量及其分布

知道每一个，不等于知道全部。这正是多维概率论存在的理由。

观测者学会了描述单个不确定量。但单个量只是世界的切片，真实数据天然是高维的。

  • 一个人的健康报告包含几十个指标。肝功能指标和血脂不是独立的——它们共享同一个代谢系统的信息。
  • 一张 \(28 \times 28\) 的灰度图像是 784 维空间中的一个点——深度学习（Ch20）本质上是在这个高维流形上做几何变换。
  • 一个投资组合包含多只股票，它们的涨跌几乎从不独立——协方差矩阵刻画了系统性风险。

Ch01 的随机变量只需回答一个问题：这个值有多大？（分布 + 期望 + 方差）。Ch02 必须同时回答三个新问题：各分量的边际分布（从全景中投影出一个维度）、分量之间的依赖结构（一个变大，另一个是否倾向于变大？）、以及给定一部分后另一部分的分布（条件分布与条件期望）。这三个问题的任何一个在 Ch01 的框架下都不存在。依赖结构是多维概率论独有的增量价值——独立性是特例，相关性是常态；边际分布是投影，联合分布才是全景。

联合分布：高维空间上的概率云

联合分布函数

\(n\) 维随机向量 \(\mathbf{X} = (X_1, X_2, \ldots, X_n)\) 是定义在同一个样本空间 \(\Omega\) 上的 \(n\) 个随机变量的组合。关键是“同一个样本空间”——多维变量的各个分量必须共享同一个随机性的源头。

联合分布函数：

\[F(x_1, \ldots, x_n) = P(X_1 \leq x_1, X_2 \leq x_2, \ldots, X_n \leq x_n)\]

这是 \(\mathbf{X}\) 落入由 \(x_1, \ldots, x_n\) 界定的左下矩形区域的概率。对于二维情况：

\[P(a < X \leq b, c < Y \leq d) = F(b,d) - F(a,d) - F(b,c) + F(a,c)\]

联合分布函数同时包含三类信息：

  • 每个分量的边际分布——单独看 \(X\) 或 \(Y\) 长什么样。
  • 分量之间的关联程度——\(X\) 大了，\(Y\) 是否也倾向于大？
  • 给定一个分量的条件分布——知道了 \(X\) 的取值，\(Y\) 的分布如何变化？

联合分布函数的结构引出一个根本问题：边际和依赖是否可以分开建模？ Sklar 定理（1959）的回答是肯定的。任何联合分布函数可以分解为

\[F(x_1,\ldots,x_p) = C\left(F_1(x_1), \ldots, F_p(x_p)\right)\]

其中 \(C: [0,1]^p \to [0,1]\) 称为 Copula——它是剥离掉所有边际分布后，剩下的纯依赖结构。换句话说，\(F_1,\ldots,F_p\) 各自决定”每个分量单独长什么样”，而 \(C\) 决定”它们之间怎么关联”。你可以独立选择边际分布族和依赖结构族——例如用正态分布做边际、用 t-Copula 做依赖，完全合法。

最简单的例子是高斯 Copula：先把每个分量通过各自的分布函数 \(F_i\) 转为 \(U(0,1)\) 均匀变量，再对这些均匀变量套用多元正态的依赖结构（通过 \(\Phi^{-1}\) 反变换到正态尺标上建立相关）。关键在于，高斯 Copula 在极端尾部假设独立——两个变量同时取极端值的概率被低估。

离散与连续的双重表达

离散场合用联合分布列：

\[p_{ij} = P(X = x_i, Y = y_j)\]

所有 \(p_{ij} \geq 0\) 且 \(\sum_i \sum_j p_{ij} = 1\)。

连续场合用联合密度函数 \(p(x,y)\)：

\[F(x,y) = \int_{-\infty}^x \int_{-\infty}^y p(u,v) \, du \, dv\]

区域 \(G\) 上的概率为 \(\iint_G p(x,y) \, dx \, dy\)。计算时需要注意两点：积分区域是 \(p(x,y)\) 的非零区域与 \(G\) 的交集；边界线（直线段）的面积为 0，是否包含不影响积分结果。

常用多维分布

多项分布：将二项分布从“成功/失败”两类推广到 \(r\) 类。\(n\) 次独立试验中，第 \(k\) 类出现 \(n_k\) 次的概率：

\[P(X_1=n_1,\ldots,X_r=n_r) = \frac{n!}{n_1! \cdots n_r!} p_1^{n_1} \cdots p_r^{n_r}\]

这是类别数据分析（Ch12）的底层分布。

二维正态分布：多元分析中最核心的连续分布。

\[p(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{ -\frac{1}{2(1-\rho^2)} \left[ \frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2} \right] \right\}\]

五个参数：\(\mu_1, \mu_2\)（中心）、\(\sigma_1^2, \sigma_2^2\)（离散度）、\(\rho\)（线性相关）。如果把这个密度画在 \(xy\) 平面上，它是一片椭圆形的概率云——中心处最厚（概率质量最集中），向外逐渐稀薄。等密度线是一族同心椭圆，椭圆的主轴方向和压扁程度由 \(\rho\) 和各方向的方差共同决定。

Code 
library(MASS)
par(mfrow = c(1, 3), mar = c(4, 4, 3, 1))
mu <- c(0, 0)
for (i in 1:3) {
  rho <- c(0, 0.5, 0.9)[i]
  Sigma <- matrix(c(1, rho, rho, 1), 2, 2)
  contour(kde2d(mvrnorm(2000, mu, Sigma)[,1], mvrnorm(2000, mu, Sigma)[,2], n = 50),
          xlab = expression(X[1]), ylab = expression(X[2]),
          main = substitute(rho == r, list(r = rho)), col = "steelblue", lwd = 1.5)
}

当 \(\rho = 0\) 时，椭圆退化为正圆，联合密度分解为两个边际密度的乘积——二元正态中，不相关等价于独立。这是一个特殊性质，并非所有分布都成立。

多元正态分布是二元正态向 \(p\) 维的推广。记 \(\mathbf{X} \sim N_p(\boldsymbol{\mu}, \Sigma)\)，其密度为：

\[p(\mathbf{x}) = \frac{1}{(2\pi)^{p/2} |\Sigma|^{1/2}} \exp\left\{ -\frac{1}{2}(\mathbf{x} - \boldsymbol{\mu})^T \Sigma^{-1} (\mathbf{x} - \boldsymbol{\mu}) \right\}\]

其中 \(\boldsymbol{\mu} \in \mathbb{R}^p\) 为均值向量，\(\Sigma\) 为 \(p \times p\) 正定协方差矩阵。二次型 \((\mathbf{x} - \boldsymbol{\mu})^T \Sigma^{-1} (\mathbf{x} - \boldsymbol{\mu})\) 是马氏距离的平方——它测量 \(\mathbf{x}\) 到中心 \(\boldsymbol{\mu}\) 的距离，单位是各主轴方向上的标准差。

多元正态的三条核心性质贯穿后续几乎所有方法：

  1. 线性变换保持正态：\(\mathbf{Y} = A\mathbf{X} + \mathbf{b} \sim N(A\boldsymbol{\mu} + \mathbf{b},\, A\Sigma A')\)。Ch05 的抽样分布和 Ch10 的线性回归都依赖这条性质。
  2. 边际与条件皆正态：任意子向量仍是多元正态；给定部分分量后，条件分布也是多元正态，且条件期望是已知分量的线性函数。这是高斯过程回归和高斯图模型的核心。
  3. 不相关等价于独立：\(\Sigma\) 非对角线全为零则各分量独立。这使得 \(\Sigma\) 的稀疏结构直接编码了条件独立——概率图模型（Ch21）的出发点。

边际分布与独立性

从联合到边际：投影

联合密度 \(p(x,y)\) 是 \(xy\) 平面上的一个曲面——某处的”高度”代表概率质量在该处的浓度。要得到 \(X\) 单独的分布（边际分布），固定 \(x\)，把所有可能的 \(y\) 上积累的概率质量累加起来：

\[p_X(x) = \int_{-\infty}^{\infty} p(x,y) \, dy, \quad p_Y(y) = \int_{-\infty}^{\infty} p(x,y) \, dx\]

这就是投影：丢掉 \(y\) 维度时，沿该方向的概率质量不会消失——它叠加到了 \(x\) 轴上。和全概率公式（Ch01）同一逻辑：穷举另一个维度的所有可能，加权求和。

独立性

多维独立性是 Ch01 独立性的直接推广——联合分布可分解为边际之积：

\[F(x,y) = F_X(x) F_Y(y) \quad \Leftrightarrow \quad p(x,y) = p_X(x) p_Y(y)\]

现实中变量几乎总有关联。独立性假设的价值在于它提供了基准模型——先假设独立，再看数据在多大程度上偏离独立，这本身就构成了判断关联强度的逻辑起点。

多维独立性还有一个关键变体：条件独立 \(X \perp\!\!\!\perp Y \mid Z\)。一旦知道 \(Z\)，\(X\) 和 \(Y\) 之间不再有信息通道。两个看起来相关的变量常常只是因为共享了一个共同原因——控制住共同原因后，相关性消失。这是因果推断（Ch22）和概率图模型（Ch21）的逻辑基础。

条件分布与条件期望：已知一部分，推断另一部分

条件分布

已知 \(Y = y\)，\(X\) 的条件密度函数：

\[p(x|y) = \frac{p(x,y)}{p_Y(y)}\]

和 Ch01 的条件概率逻辑一致：观测到 \(Y=y\) 后，概率空间从 \(\mathbb{R}^2\) 塌缩到直线 \(Y=y\) 上，\(p(x|y)\) 是这条直线上 \(X\) 的密度。

二维正态的条件分布仍是一维正态：\(X \mid Y=y \sim N\left(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2), \; \sigma_1^2(1-\rho^2)\right)\)。条件期望是 \(y\) 的线性函数，条件方差小于边际方差（因为 \(Y\) 提供了信息）。这与 Ch01 的条件概率共享同一套逻辑骨架——只是求和换成了积分。

条件期望：统计学最核心的预测工具

观测者知道了 \(Y\) 的取值。他必须更新对 \(X\) 的判断——用的不再是 \(X\) 的边际分布，而是条件分布。条件期望 \(E(X \mid Y=y)\) 是他能做出的、在均方误差意义下最优的预测。

\[E(X \mid Y=y) = \begin{cases} \sum_i x_i \, P(X=x_i \mid Y=y) & \text{离散} \\ \int_{-\infty}^{\infty} x \, p(x|y) \, dx & \text{连续} \end{cases}\]

条件期望 \(E(X \mid Y=y)\) 是 \(y\) 的函数——不同的 \(y\) 给出不同的期望值。把这个函数记为 \(g(y)\)，则 \(E(X \mid Y)\) 本身是一个随机变量——它是 \(Y\) 的变换，其随机性来源于 \(Y\) 的随机性。

核心洞察：\(E(Y \mid X)\) 是“已知 \(X\) 时，在最小均方误差（MSE）意义下对 \(Y\) 的最佳预测”。这个事实贯穿全书：

  • Ch10 回归分析：用线性函数近似 \(E(Y \mid X)\)
  • Ch17 机器学习：任何监督学习的目标函数都是 \(E(Y \mid X)\) 的某种估计
  • Ch20 深度学习：神经网络是用梯度下降逼近高度非线性的 \(E(Y \mid X)\)

这是全书被引用次数最多的公式。

重期望公式：分而治之

\[E(X) = E[E(X \mid Y)]\]

推导：由条件期望的定义，

\[E[E(X \mid Y)] = \int E(X \mid Y=y) \, p_Y(y) \, dy = \int \left[ \int x \, p(x|y) \, dx \right] p_Y(y) \, dy = \iint x \, p(x,y) \, dx \, dy = E(X)\]

先按 \(Y\) 的取值把整个空间切成若干片，在每一片上求 \(X\) 的均值，再对片与片之间加权平均——权重是每片的概率。这不仅是计算技巧，更是一种世界观：如果一个问题太大、太复杂，就找一个相关的 \(Y\) 把它拆开，逐个击破。

重期望公式最常见的应用是将方差分解为两部分。推导如下：

\[ Var(X) = E[(X - E[X])^2]= E[((X - E[X|Y]) + (E[X|Y] - E[X]))^2] \]

第一项 \(X - E[X|Y]\) 是”组内残差”（\(X\) 偏离其组均值的程度），第二项 \(E[X|Y] - E[X]\) 是”组间偏差”（组均值偏离总均值的程度）。两者平方后交叉项的期望为零——由重期望公式保证：

\[E[(X - E[X|Y])(E[X|Y] - E[X])] = E\left[(E[X|Y] - E[X]) \cdot E[X - E[X|Y] \mid Y]\right] = 0\]

因此得到方差分解公式：

\[\text{Var}(X) = E[\text{Var}(X \mid Y)] + \text{Var}(E[X \mid Y])\]

第一项是”组内方差的平均”（不可由 \(Y\) 解释的变异），第二项是”组间均值的方差”（可由 \(Y\) 解释的变异）。这是 Ch10 的 \(R^2\) 分解和 Ch09 的 EM 算法 E 步中反复出现的结构。

多维随机变量函数的分布

已知 \((X_1, \ldots, X_n)\) 的联合分布，如何求出 \(Y = g(X_1, \ldots, X_n)\) 的分布？

卷积：独立随机变量之和的分布

当 \(X\) 和 \(Y\) 独立时，\(Z = X + Y\) 的分布为什么需要一个新的运算？因为 \(Z = k\) 可以由多种 \((X=i, Y=k-i)\) 的组合实现——每一种组合是一条独立的路径，其概率由独立性保证为 \(P(X=i)P(Y=k-i)\)，而 \(Z=k\) 的总概率需要沿所有这些路径累加。这就是卷积的结构来源。

记 \(f_X(i) = P(X=i)\)，\(f_Y(j) = P(Y=j)\)，定义两者的卷积 \((f_X * f_Y)(k)\) 为：

\[(f_X * f_Y)(k) = \sum_{i} f_X(i) \, f_Y(k - i)\]

即 \(P(Z = k) = (f_X * f_Y)(k)\)。几何上，这是将 \(f_Y\) 翻转后沿 \(f_X\) 滑动、逐点相乘再求和——因此称为”卷积”。

连续版本的卷积将求和换为积分：

\[(f_X * f_Y)(z) = \int_{-\infty}^{\infty} f_X(z-y) \, f_Y(y) \, dy = \int_{-\infty}^{\infty} f_X(x) \, f_Y(z-x) \, dx\]

卷积是全书最重要的运算之一。Ch03 的特征函数将卷积转化为乘法——这是证明 CLT 的核心技巧；Ch10 的线性回归中，估计量是数据的线性卷积；Ch20 的卷积神经网络直接在像素网格上做离散卷积。

可加性：若同一分布族的两个独立随机变量的和仍属于该族，则称该族具有可加性。

  • 泊松分布：\(P(\lambda_1) * P(\lambda_2) = P(\lambda_1 + \lambda_2)\)
  • 二项分布（同 \(p\)）：\(b(n,p) * b(m,p) = b(n+m,p)\)
  • 正态分布：\(N(\mu_1,\sigma_1^2) * N(\mu_2,\sigma_2^2) = N(\mu_1+\mu_2, \sigma_1^2+\sigma_2^2)\)
  • 伽马分布（同 \(\lambda\)）：\(Ga(\alpha_1, \lambda) * Ga(\alpha_2, \lambda) = Ga(\alpha_1+\alpha_2, \lambda)\)

具有可加性的分布族在独立和运算下封闭——正态分布是唯一在任意线性组合下都保持封闭的分布。可加性是 CLT 的前奏：当你不断对独立同分布变量做卷积，分布形状会逐渐演化，而正态分布是这个演化过程的唯一不动点（Ch03）。

最大值与最小值的分布

\(n\) 个独立同分布变量 \(X_i \sim F\)，令 \(Y = \max\{X_1, \ldots, X_n\}\)，\(Z = \min\{X_1, \ldots, X_n\}\)：

\[F_Y(y) = [F(y)]^n, \quad F_Z(z) = 1 - [1 - F(z)]^n\]

最大值分布是极值理论（Ch13 非参数方法相关）的基础，在异常检测中有直接应用。

变量变换法（雅可比方法）

设 \(\mathbf{Y} = \mathbf{g}(\mathbf{X})\) 是一一变换，其逆变换为 \(\mathbf{X} = \mathbf{h}(\mathbf{Y})\)，则：

\[p_{\mathbf{Y}}(\mathbf{y}) = p_{\mathbf{X}}(\mathbf{h}(\mathbf{y})) \cdot |\det J|\]

其中 \(J = \partial \mathbf{h}(\mathbf{y}) / \partial \mathbf{y}\) 是雅可比矩阵，\(|\det J|\) 是变换对体积的伸缩因子——从 \(dx\) 到 \(dy\) 的坐标变换，概率质量必须守恒。

多元正态变量经线性变换后仍为多元正态变量。这是多元正态分布最重要的性质，也是它在多元分析（Ch21）和概率图模型中不可替代的原因。

两个实用的积/商公式。在金融中，收益率之积对应复利累积；在比率估计中，两个随机量的比值自然出现。令 \(U = XY\) 和 \(U = X/Y\)：

\[p_U(u) = \int_{-\infty}^{\infty} p_X\left(\frac{u}{v}\right) p_Y(v) \, \frac{1}{|v|} \, dv\]

\[p_U(u) = \int_{-\infty}^{\infty} p_X(uv) \, p_Y(v) \, |v| \, dv\]

协方差与相关系数

协方差矩阵：高维分布的结构骨架

协方差是两个随机变量联合偏离各自期望的程度的期望：

\[\text{Cov}(X,Y) = E[(X - EX)(Y - EY)] = E(XY) - E(X)E(Y)\]

\(\text{Cov}(X,Y) > 0\) 意味着 \(X\) 和 \(Y\) 倾向于同向偏离各自的均值；\(\text{Cov}(X,Y) < 0\) 意味着反向偏离。

若 \(X\) 与 \(Y\) 独立，则 \(\text{Cov}(X,Y) = 0\)。但反之不然——协方差只捕获线性关联。两个有完美的非线性关系（如 \(Y = X^2\)）的变量，协方差可以为零。

观测者面对的不再是单变量的波动，而是一个 \(p\) 维数据云。他在问：这个云最伸展的方向在哪里？各方向之间如何彼此牵制？协方差矩阵 \(\Sigma\) 回答的就是这些问题。对于 \(n\) 维随机向量 \(\mathbf{X}\)：

\[\Sigma = \text{Cov}(\mathbf{X}) = E[(\mathbf{X} - E\mathbf{X})(\mathbf{X} - E\mathbf{X})'] = \begin{pmatrix} \text{Var}(X_1) & \text{Cov}(X_1,X_2) & \cdots & \text{Cov}(X_1,X_n) \\ \text{Cov}(X_2,X_1) & \text{Var}(X_2) & \cdots & \text{Cov}(X_2,X_n) \\ \vdots & \vdots & \ddots & \vdots \\ \text{Cov}(X_n,X_1) & \text{Cov}(X_n,X_2) & \cdots & \text{Var}(X_n) \end{pmatrix}\]

\(\Sigma\) 是对称的半正定矩阵，对角线是各分量的方差，非对角线是两两间的协方差。\(\Sigma\) 不仅是一张数字表——它是一个线性算子，刻画了数据云的形状。

\(\Sigma\) 是一个线性算子。对它做谱分解 \(\Sigma = U \Lambda U^T\)，就是在问：数据云最自然伸展的方向在哪里？答案藏在特征系统里——\(U\) 的列是互相正交的主轴方向（特征向量），\(\Lambda\) 的对角线是沿各主轴方向的方差（特征值）。PCA（Ch21）不是一套算法步骤，而是一次坐标旋转：将数据转到它自己的主轴坐标系下。

沿这些主轴测量距离，就是马氏距离——以 \(\Sigma^{-1}\) 为度规，方差大的方向上”宽容”，方差小的方向上”严格”。这正是多元异常检测和 Hotelling \(T^2\) 检验（Ch07、Ch21）的几何直觉。

在主轴坐标系下观察 \(\Lambda\)，\(\Sigma\) 的三个标量摘要变得一目了然：\(\operatorname{tr}(\Sigma) = \sum \lambda_i\)（数据云的总波动能量）、\(|\Sigma| = \prod \lambda_i\)（广义方差——置信椭球的超体积 \(\propto \sqrt{|\Sigma|}\)）、\(\operatorname{rank}(\Sigma)\)（真正有波动的维度数）。当 \(p > n\) 时，\(\Sigma\) 必然不满秩——这是高维统计的结构性约束。

如果从主轴坐标系再往前推一步——把所有主轴统一归一化，就得到一个各向同性的球对称噪声。要把这个白噪声”还原”为原始数据云的形状，你需要乘以一个矩阵 \(L\)。Cholesky 分解 \(\Sigma = LL^T\) 给出的 \(L\) 正是这个”形状恢复矩阵”：\(Z \sim N(0,I) \;\Rightarrow\; LZ + \mu \sim N(\mu, \Sigma)\)。这一”用确定性变换将噪声映射为目标分布”的思路，是变分自编码器重参数化技巧（Ch20）的原型。

精度矩阵 \(\Sigma^{-1}\) 在协方差矩阵的应用中占有特殊地位：其 \((i,j)\) 元素为零，等价于在给定其他变量的条件下 \(X_i\) 与 \(X_j\) 条件独立。这意味着 \(\Sigma^{-1}\) 的稀疏结构直接编码了变量间的条件独立关系——概率图模型（Ch21）正是围绕这一事实构造的。

方差运算

\[\text{Var}\left(\sum_{i=1}^n a_i X_i\right) = \sum_{i=1}^n a_i^2 \text{Var}(X_i) + 2\sum_{i<j} a_i a_j \text{Cov}(X_i, X_j)\]

当 \(X_i\) 互不相关时，交叉项消失——方差可加。但当它们相关时，总方差可能被放大（正相关）或被对冲（负相关）。分散投资降低风险的统计学原理，正在于此。

\(n\) 个独立同分布变量的算术平均的方差：

\[\text{Var}\left(\frac{1}{n}\sum_{i=1}^n X_i\right) = \frac{\sigma^2}{n}\]

这 \(1/n\) 的衰减是大数定律（Ch03）的前奏——平均让方差缩小，让估计越来越准。

相关系数：消除量纲的纯关联

\[\text{Corr}(X,Y) = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)\,\text{Var}(Y)}} = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}\]

由施瓦兹不等式保证：\(-1 \leq \text{Corr}(X,Y) \leq 1\)。

\(\text{Corr} = \pm 1\) 的充要条件是 \(X\) 与 \(Y\) 之间几乎处处有精确的线性关系。\(\text{Corr} = 0\) 只意味着”没有线性关系”，不意味”没有关系”。

互信息：相关性的信息论度量

相关系数 \(\rho\) 度量的是线性相关强度。互信息 \(I(X;Y)\) 度量的是任意依赖关系的强度：

\[I(X;Y) = H(Y) - H(Y|X) = H(X) - H(X|Y)\]

它回答的问题是：知道 \(X\)，能让 \(Y\) 的不确定性减少多少？ 当 \(X\) 和 \(Y\) 独立时，\(I(X;Y) = 0\)；当 \(Y\) 是 \(X\) 的确定性函数时，\(I(X;Y) = H(Y)\)。

互信息、联合熵、条件熵之间的关系链：

\[I(X;Y) = H(X) + H(Y) - H(X,Y)\]

这在 Ch18 决策树（信息增益 = 互信息）和 Ch20 表示学习（信息瓶颈）中反复出现。

相关系数 \(\rho\) 是二元正态分布的第五个参数，也是整个回归分析（Ch10）的起点——\(\rho^2\) 就是 \(R^2\)（决定系数）的母体，度量”用 \(X\) 解释 \(Y\) 的变异”的解释比例。

  • 核心问题：如何描述多个随机变量的联合行为？联合分布给出全景（Copula 揭示边际与依赖可解耦），边际分布给出剪影，条件分布给出局部，协方差矩阵（作为线性算子）刻画数据云的形状与方向。

深度思考：

  1. 条件期望 \(E(Y \mid X)\) 是“已知 \(X\) 时对 \(Y\) 的最佳预测”。在什么意义上“最佳”？如果我们换用绝对误差（MAE）而非平方误差（MSE），最佳预测还是条件期望吗？——中位数在 MAE 下的最优性和条件期望在 MSE 下的最优性，揭示了一个深刻的原理：损失函数定义了“好”的含义（Ch06 和 Ch17 将反复回到这个主题）。

  2. 协方差和相关系数只度量线性关联。如果 \(Y = \sin(X)\)，相关系数可以接近零。这是相关系数的缺陷，还是线性性的力量？——在什么场景下，关注线性关联是足够的？

  3. 协方差矩阵 \(\Sigma\) 的维度是 \(p \times p\)，参数数量是 \(O(p^2)\)。当 \(p = 1000\) 时，你需要估计约 50 万个参数——而样本量可能只有几百。这个 \(p > n\) 的困境是高维统计的核心挑战。在你的直觉中，有什么方法可以应对？

  4. 高斯 Copula 在 2008 年金融危机前被广泛用于 CDO 定价——它假设各抵押贷款的违约在尾部是独立的。为什么这个假设在危机中崩溃了？如果一个数据科学家今天要为极端事件建模，他应该优先考虑什么依赖结构？提示：尾部依赖的有无，决定了”百年一遇”的事件是否倾向于扎堆出现。

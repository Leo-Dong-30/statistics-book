统计量与抽样分布

你有一组数据。你相信它背后藏着一个更深的真相——某个未知的参数 \(\theta\)。你的任务，是从可观测的 \(n\) 个数字逆推出那个不可见的源头。本章建立这个”逆推”所需的全部数学地基。

从概率论到数理统计

概率论与数理统计是一对镜像。

概率论回答：已知参数 \(\theta\)，数据 \(X\) 会呈现什么行为？这是演绎——从原因推结果。你设定骰子是均匀的（\(p = 1/6\)），然后推导连掷 10 次出现 3 次六点的概率。

数理统计回答相反的提问：已知观测数据 \(X\)，\(\theta\) 最可能是什么？这是归纳——从结果反推原因。你拿到了 100 次掷骰子的记录，需要推断这枚骰子是否均匀。

\[\text{概率论（演绎）}: \theta \longrightarrow X \qquad \text{数理统计（归纳）}: X \longrightarrow \theta\]

这两个学科共享同一套数学对象，但方向相反。概率论以分布函数 \(F(x;\theta)\) 为武器从参数推向数据；数理统计以样本为起点从数据追溯参数。

在这个框架下，总体就是一个分布——当我们说”从总体中抽样”，精确的数学含义是”从分布 \(F\) 中独立地抽取 \(n\) 个随机变量”。这 \(n\) 个随机变量构成的向量 \((X_1, \ldots, X_n)\) 是 \(n\) 维空间中的一个点，其联合分布为 \(\prod_{i=1}^n F(x_i)\)。

但 \(n\) 可能很大——几百、几千、几百万。而控制这些数据的参数 \(\theta\) 往往只是区区几个实数。信息高度冗余。统计量正是为消除这个冗余而定义的：

\[T = T(X_1, \ldots, X_n): \mathbb{R}^n \to \mathbb{R}^k \quad (k \ll n)\]

统计量是不含任何未知参数的样本函数。它将 \(n\) 维的原始数据压缩为 \(k\) 维的摘要——样本均值是一个统计量（\(k=1\)），样本均值和样本方差在一起是两个统计量（\(k=2\)）。统计量的分布称为抽样分布——它刻画了这个压缩过程本身的随机波动规律。

数理统计的全部逻辑起点在一个问题里：我们能否只通过一个低维统计量，就完整地”读到”样本中关于 \(\theta\) 的全部信息？ 如果答案是肯定的，所有后续推断——估计、检验、预测——都可以在这个压缩后的低维空间中进行。这就是下一层的核心命题。

充分统计量：信息无损的压缩

样本 \((X_1, \ldots, X_n)\) 是信息的原始载体。统计量 \(T\) 是一个压缩映射——它扔掉了一些东西。问题是：扔掉的，是噪声还是信号？

充分统计量（Sufficient Statistic）的概念精确地回答了这个问题。设样本来自分布 \(F(x;\theta)\)，统计量 \(T\) 称为 \(\theta\) 的充分统计量，如果给定 \(T\) 的取值后，样本的条件分布不再依赖于 \(\theta\)：

\[P(X_1, \ldots, X_n \mid T = t; \theta) = P(X_1, \ldots, X_n \mid T = t)\]

这个定义的物理直觉是：一旦你知道了 \(T\)，原始样本中就不再含有任何关于 \(\theta\) 的额外信息。\(T\) 已经”榨干”了样本中与 \(\theta\) 相关的一切——剩余的只是纯噪声。从信息论的角度看，\(T\) 是对样本的无损压缩：降维的同时，截断了所有与 \(\theta\) 无关的方向，保留了包含 \(\theta\) 信息的子空间。

全样本本身永远是充分的——但它没有压缩。最小充分统计量（Minimal Sufficient Statistic）追求的是压缩比的极限：在所有充分统计量中，它是”最粗”的那个——将数据压缩到了”刚好不丢失 \(\theta\) 信息”的临界点。任何进一步的压缩都会切断通往 \(\theta\) 的信息通道。

因子分解定理（Fisher–Neyman Factorization Theorem）提供了判定充分统计量的实用工具。设样本的联合密度为 \(f(x_1,\ldots,x_n;\theta)\)，则 \(T\) 为充分统计量的充要条件是存在两个函数 \(g\) 和 \(h\)，使得：

\[f(x_1,\ldots,x_n;\theta) = g(T(x_1,\ldots,x_n), \theta) \cdot h(x_1,\ldots,x_n)\]

其中 \(g\) 依赖于样本仅通过 \(T\) 的取值，\(h\) 与 \(\theta\) 无关。这个分解的结构信息极其清晰：\(g\) 是似然函数中”与 \(\theta\) 有关的部分”——它完全经由 \(T\) 这个瓶颈流动；\(h\) 是纯数据内禀因子——它在推断 \(\theta\) 时可以忽略。

因子分解定理揭示了充分性的本质不在于 \(T\) 本身是什么，而在于似然函数的因子化结构。一旦 \(f(x;\theta)\) 可以写成”仅依赖 \(T\) 的部分 × 与 \(\theta\) 无关的部分”，\(T\) 就自动成为充分统计量。这个结构的极端体现是指数型分布族。

指数型分布族与充分统计量之间存在天然的对应关系。Ch01 中引入了指数族的一般形式：

\[f(x;\theta) = h(x)\exp\left(\eta(\theta)^T T(x) - A(\theta)\right)\]

对比因子分解定理的 \(g(T,\theta) \cdot h(x)\) 结构，可以立刻读出：\(T(x)\) 就是 \(\theta\) 的充分统计量。指数族自动为你提供了充分统计量——这就是为什么正态分布中 \(\sum x_i\) 是 \(\mu\) 的充分统计量，泊松分布中同样的 \(\sum x_i\) 是 \(\lambda\) 的充分统计量。Ch11（广义线性模型）中，GLM 将响应变量限定在指数族内，正是为了继承这套天然的充分统计量结构。

经验分布函数

在进入具体的参数推断之前，一个更根本的问题值得追问：我们能不能根本不知道 \(F(x)\) 的函数形式，却依然逼近它？

经验分布函数（Empirical Distribution Function）给出了肯定的回答。设样本有序排列为 \(x_{(1)} \leq x_{(2)} \leq \cdots \leq x_{(n)}\)，定义：

\[F_n(x) = \frac{\#\{i: x_i \leq x\}}{n} = \begin{cases} 0, & x < x_{(1)} \\ k/n, & x_{(k)} \leq x < x_{(k+1)} \\ 1, & x \geq x_{(n)} \end{cases}\]

\(F_n(x)\) 是一个阶梯函数——在每个数据点处向上跳 \(1/n\)。它不假设任何参数形式，只忠实地用样本的比例去近似总体的累积概率。

格利文科定理（Glivenko–Cantelli Theorem）给出了这个逼近的数学担保：

\[P\left(\sup_{-\infty < x < \infty} |F_n(x) - F(x)| \to 0\right) = 1, \quad n \to \infty\]

经验分布函数一致地收敛于真实分布——不只是逐点的，而是在整个实数轴上，最大偏差也趋于零。这个定理是统计学中最被低估的深刻结果：你不需要知道真实分布的任何函数形式。只要不断抽样，样本的经验分布就会一致地逼近它。 这是所有非参数统计方法（Ch13）的逻辑许可证——你可以直接从数据本身学习分布的形状，而不必假设它来自正态、指数或任何特定的参数族。

格利文科定理是”渐近承诺”——它说 \(n \to \infty\) 时一切完美。但现实中的 \(n\) 永远是有限的。Ch04 的 DKW 不等式给出了有限样本下的非渐近界：\(P(\sup_x |F_n(x) - F(x)| > \epsilon) \leq 2e^{-2n\epsilon^2}\)。它不仅保证”最终会准”，而且对给定的 \(n\) 和 \(\epsilon\)，失败概率以指数速率衰减。从格利文科到 DKW，统计学完成了从哲学到工程的跃迁。

次序统计量与边缘分布

经验分布函数是对分布整体的逼近。但有时观测者只关心分布的边缘——最悲观的底线在哪里？最高可能触及何处？中位观测者处于什么水平？

次序统计量将样本从小到大排列：\(x_{(1)} \leq x_{(2)} \leq \cdots \leq x_{(n)}\)。\(x_{(k)}\) 称为第 \(k\) 个次序统计量。\(x_{(1)}\) 是样本最小值（“短板”），\(x_{(n)}\) 是样本最大值（“天花板”），中位数则落在中间。

第 \(k\) 个次序统计量的精确分布有一个优雅的概率解释：事件 \(\{x_{(k)} \leq x\}\) 等价于”至少有 \(k\) 个样本点落在 \((-\infty, x]\) 内”。由此可以导出密度函数：

\[p_{(k)}(x) = \frac{n!}{(k-1)!(n-k)!} [F(x)]^{k-1} [1 - F(x)]^{n-k} p(x)\]

其中 \(p(x)\) 是总体的密度函数、\(F(x)\) 是分布函数。三个因子各司其职：组合数来自”\(n\) 个中选 \(k-1\) 个在左边、\(n-k\) 个在右边”的排列方式；\(F^{k-1}\) 是左边 \(k-1\) 个都在 \(x\) 之下的概率；\((1-F)^{n-k}\) 是右边 \(n-k\) 个都在 \(x\) 之上的概率；\(p(x)\) 是第 \(k\) 个恰好落在 \(x\) 处的概率密度。

两端极值的情况尤为简洁：\(p_{(1)}(x) = n[1-F(x)]^{n-1} p(x)\)，\(p_{(n)}(x) = n[F(x)]^{n-1} p(x)\)。

极值的渐近行为与中心极限定理（研究均值）形成了互补的认知维度。CLT 告诉我们”典型观测者的平均”趋向正态分布；极值理论（Extreme Value Theory）告诉我们”最极端的观测者”趋向 Gumbel、Frechet 或 Weibull 分布——取决于原始分布的尾部厚度。均值与极值，一起构成了对总体从中心到边缘的完整视野。

在次序统计量中，中位数具有特殊的稳健性地位。当数据含有离群值时，均值可能被严重扭曲，但中位数几乎不受影响——一个数据点的改变最多只能让中位数跳到相邻的次序统计量。这一稳健性使得中位数回归（分位数回归的特例）在重尾数据的场景中成为均值的有效替代。

三大抽样分布：推断工具

前四层完成了从”数据”到”信息”的抽象建构。现在，为了在正态总体的假设下做具体的统计推断（Ch06 的区间估计、Ch07 的假设检验），我们需要三个精确的抽样分布作为”度量衡”。

\(\chi^2\) 分布——衡量变异。 设 \(Z_1, \ldots, Z_n \overset{\text{iid}}{\sim} N(0,1)\)，定义：

\[\chi^2 = Z_1^2 + Z_2^2 + \cdots + Z_n^2 \sim \chi^2(n)\]

其密度函数为 \(\text{Gamma}(n/2, 1/2)\)：\(f(x; n) = \frac{1}{2^{n/2}\Gamma(n/2)} x^{n/2-1} e^{-x/2},\; x > 0\)。\(E[\chi^2] = n\)，\(\text{Var}(\chi^2) = 2n\)。\(\chi^2\) 分布自然地成为样本方差推断的枢纽：来自正态总体的样本，\(\frac{(n-1)s^2}{\sigma^2} \sim \chi^2(n-1)\)。

\(t\) 分布——对未知的妥协。 实际应用中，\(\sigma\) 几乎总是未知的。当我们用样本标准差 \(s\) 代替 \(\sigma\) 来标准化样本均值时，分子是标准正态 \(Z\)，分母是 \(\sqrt{\chi^2/\nu}\)——这个比值不再服从正态分布，而是 \(t\) 分布：

\[t = \frac{Z}{\sqrt{\chi^2(n)/n}} \sim t(n)\]

\(t\) 分布的密度比标准正态的峰更低、尾部更厚——这是用 \(s\) 估计 \(\sigma\) 带来的额外不确定性的数学表达。自由度 \(n\) 越小（样本越少），这种不确定性越显著，尾部越厚；\(n \to \infty\) 时，\(s \to \sigma\)，\(t\) 分布收敛于标准正态。

\(t\) 分布的发现（Gosset, 1908，以”Student”为笔名）打破了正态分布一统天下的局面，开创了小样本统计推断的新纪元。它的哲学含义是坦诚的：我承认我不知道 \(\sigma\)，我为此付出厚尾的代价。

\(F\) 分布——衡量变异之比。 比较两个独立正态总体的方差时，两个标准化后的样本方差之比服从 \(F\) 分布：

\[F = \frac{\chi^2(m)/m}{\chi^2(n)/n} \sim F(m, n)\]

若两个总体的方差相等（\(\sigma_1^2 = \sigma_2^2\)），则 \(F = s_x^2 / s_y^2 \sim F(m-1, n-1)\)。\(F\) 分布是 ANOVA（Ch11）和模型比较中 \(F\) 检验的数学基础——它衡量的是”模型解释的变异”与”无法解释的随机噪音”的比值。

三大分布的内在统一并非各自孤立：\(t^2(n) \sim F(1, n)\)——\(t\) 分布是 \(F\) 分布在分子自由度为 1 时的开方；\(F\) 分布是两个 \(\chi^2\) 分布之比；\(\chi^2\) 分布是独立标准正态的平方和。三者共同构成了正态总体下所有精确推断的分布基础。

样本统计量的基本性质

有了抽样分布作为工具，现在回到最常用的两个统计量，总结它们的期望、方差以及在正态总体下的精确分布。

样本均值 \(\bar{X} = \frac{1}{n}\sum X_i\) 是对总体均值 \(\mu\) 最自然的估计：

\[E(\bar{X}) = \mu, \quad \text{Var}(\bar{X}) = \frac{\sigma^2}{n}\]

两个性质共同构成 LLN 和 CLT（Ch03）的微观表达：期望等于真值（无偏性），方差以 \(1/n\) 衰减（相合性）。若总体为 \(N(\mu, \sigma^2)\)，则 \(\bar{X} \sim N(\mu, \sigma^2/n)\) 精确成立；若总体非正态但 \(n\) 较大，则由 CLT 保证 \(\bar{X} \overset{\text{approx}}{\sim} N(\mu, \sigma^2/n)\)。

样本方差 \(s^2 = \frac{1}{n-1}\sum (X_i - \bar{X})^2\) 的分母使用 \(n-1\) 而非 \(n\)，这是数理统计中最经典的人为修正：

\[E(s^2) = \sigma^2\]

分母用 \(n\) 的版本（\(s_n^2\)）有偏：\(E(s_n^2) = \frac{n-1}{n}\sigma^2 < \sigma^2\)。偏的来源是 \(\bar{X}\) 本身也消耗了一个自由度来估计 \(\mu\)——\(n\) 个偏差 \((X_i - \bar{X})\) 之间存在一个线性约束 \(\sum (X_i - \bar{X}) = 0\)，因此独立偏差只有 \(n-1\) 个。除以 \(n-1\) 是对”用数据估计了 \(\mu\) 之后再估计 \(\sigma^2\)“这一双重使用数据行为的补偿。

样本矩是期望和方差的自然推广。样本 \(k\) 阶原点矩 \(a_k = \frac{1}{n}\sum X_i^k\) 估计总体 \(E(X^k)\)；样本 \(k\) 阶中心矩 \(b_k = \frac{1}{n}\sum (X_i - \bar{X})^k\) 估计总体 \(E[(X-\mu)^k]\)。由此派生出的样本偏度 \(\hat{\beta}_s = b_3 / b_2^{3/2}\) 和样本峰度 \(\hat{\beta}_k = b_4 / b_2^2 - 3\) 分别捕捉数据的不对称性和尾部厚度——它们在 Ch07 的正态性检验和 Ch13 的数据探索中反复出现。

从样本均值到样本方差到样本矩，统计量始终在回答同一个问题：如何从 \(n\) 个数字中提炼出尽可能少、但又能充分代表总体特征的摘要？ 充分统计量是这一追问的极限答案——当它存在时，你只需要一个或几个数字就能完全替代原始样本。即使充分统计量不存在，我们仍然可以通过经验分布函数（整体逼近）和次序统计量（边缘刻画）获取对总体的全面认知。这六层逻辑合在一起，构成了经典统计推断不可动摇的地基。

深度思考：

  1. 充分统计量定义为”给定 \(T\) 后，样本的条件分布与 \(\theta\) 无关”。这一定义的操作意义是什么？——如果你只拿到 \(T\) 的取值而丢失了原始数据，你是否在数学上”损失了任何关于 \(\theta\) 的东西”？因子分解定理如何将这个问题转化为纯粹的代数判定？

  2. 格利文科定理说经验分布一致收敛于真实分布。相比之下，LLN 只给出逐点收敛（\(\bar{X}_n \to \mu\)）。“一致收敛”比”逐点收敛”强在哪里？为什么这个强度对非参数统计方法的合法性至关重要？

  3. \(t\) 分布的厚尾来自”用 \(s\) 估计 \(\sigma\) 的额外不确定性”。为什么用 \(\sigma\)（已知）时得到正态分布，用 \(s\)（估计）时得到 \(t\) 分布？从抽样分布的推导过程中，这一步的数学机制是什么？——提示：比较 \(Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}}\) 和 \(t = \frac{\bar{X} - \mu}{s/\sqrt{n}}\) 中分母的随机性。

  4. 指数族提供了充分统计量的”自动生成机制”。在什么意义上，指数族是唯一具有此性质的分布族？——提示：思考因子分解定理的结构与指数族形式的对应关系，以及 Darmois–Pitman–Koopman 定理的断言。
